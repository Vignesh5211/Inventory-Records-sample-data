# Inventory-Records-sample-data
Built an Excel-based inventory tracking system to monitor stock levels, calculate stock value, and analyze top-performing products.


Key Features:

Automated stock calculation using SUM & IF

Stock turnover analysis

Pivot table summaries

Top 3 product dashboard

Tools Used:
Microsoft Excel

Outcome:
Improved visibility of inventory levels and cost tracking.
